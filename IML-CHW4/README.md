# README for IML CHW4

## Project Overview

This assignment is part of the *Introduction to Machine Learning* course.  
The objective of this homework is to explore neural networks and advanced optimization techniques, focusing on deep learning fundamentals and practical implementation.

The assignment emphasizes understanding how neural networks learn, how gradients propagate, and how optimization affects performance.

This homework is centered around implementing and analyzing:

- Multi-Layer Perceptrons (MLP)
- Backpropagation
- Optimization strategies
- Regularization techniques

---

## Structure of the Assignment

### Part 1 – Neural Network Fundamentals

Main steps:

- Implement forward propagation
- Define activation functions
- Compute loss functions
- Analyze model capacity

Key learning concepts:

- Linear transformations
- Non-linear activation functions
- Model expressiveness
- Overfitting vs underfitting

---

### Part 2 – Backpropagation and Optimization

Main steps:

- Derive gradients analytically
- Implement backpropagation manually
- Train the network using Gradient Descent
- Monitor loss convergence

Key learning concepts:

- Chain rule in deep learning
- Gradient flow
- Vanishing / exploding gradients
- Learning rate sensitivity

---

### Part 3 – Regularization and Performance Analysis

Main steps:

- Implement regularization (L2 or dropout if included)
- Compare training vs validation performance
- Visualize decision boundaries
- Analyze generalization behavior

Key learning concepts:

- Bias–variance trade-off
- Generalization
- Model stability
- Hyperparameter tuning

---
